Q. Discuss the scope of var, let, and const

Var: Var is a global scope. Var can access from anywhere. var create variables that can be reassigned another value, can declare same variable that will change the value of first variable.

Let: Let is block scope. If declared let inside of function that will not hoisting. we can not access that variable from outside. let create variables that can be reassigned another value but Can not declare as a same variable.

Const: const is also a block scope. const create variable that can not be reassigned another value.

Q. Tell us the use cases of null and undefined

Null: Null means an empty value. Declare variable as a null means no value.

Undefined:When variable declare but value not assigned, and try to access that will return undefined

Q. What do you mean by REST API?

An API(Application programming Interface) is the way of communication application and devices. A REST API is an API that confirms to the design principles of the REST, or representational state transfer architectural style.For this reason, REST APIs are sometimes referred to RESTful APIs.
