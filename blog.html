Q. Discuss the scope of var, let, and const

Var: Var is a global variable. Var can access from anywhere.

Let: Let is block scope. if we declare a variable with let in the function, we can also declare with the same variable other side of scope.

Const: const is also a block scope. but we can not re assign it.

Q. Tell us the use cases of null and undefined

null:When we don't declare any thing and want to access that will return us null

undefined:When something declare but not provided value . and try to access that will return undefined

Q. What do you mean by REST API?

An API(Application programming Interface) is the way of communication application and devices. A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style.For this reason, REST APIs are sometimes referred to RESTful APIs.
